Virtual codechef contests

This website will allow you to virtually take part in codechef contests that have already been conducted on codechef.
Since codechef doesn't provide the option for virtual participation, We made this website.
The key components shall be:

  • Virtual participaion. Time limited contest with proper markings.
  • Login functionality so that users can track their progress.
  • Fetching submissions during contest to simulate competitive environment
    • Real time ranking based on original contest submissions.
    • Final ranking if user had taked part in the original contest.
  • Displaying the problem statement without tags and editorial links. (Not top priority; Determine what's the best way to do this.)
